# html/html-to-docx-converter

## Purpose

Convert operation using Aspose.HTML plugin API.

**Canonical URL**: [https://products.aspose.net/html/html-to-docx-converter/](https://products.aspose.net/html/html-to-docx-converter/)

## NuGet Package

`Aspose.HTML` (version managed centrally in `Directory.Packages.props`; version 24.12.0 proven)

## Prerequisites

- .NET 8.0 SDK or later
- NuGet package `Aspose.HTML` (restored automatically by `dotnet restore`)

## Build & Run

```bash
dotnet restore
dotnet build
dotnet run
```

## Expected Output

convert result (.docx)

| File | Description |
